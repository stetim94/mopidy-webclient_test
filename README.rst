Installation
============

Install by running::

    pip install Mopidy-webclient_test

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============

Mopidy-webclient_test to your Mopidy configuration file::
